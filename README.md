# AI-Powered Code Understanding & Mentorship Platform

A "Premium" AI Mentor that explains codebases, provides feedback, and helps you prepare for interviews.

## Features
- **Project Analysis**: Upload Zip or Clone GitHub Repos.
- **AI Mentorship**: Explains code in English, Hindi, or Hinglish.
- **Smart Quiz Generator**: Tests your knowledge on specific files.
- **Interactive UI**:
    - **Copy Code**: Easily copy snippets.
    - **Mentor Chat**: Clear history and switch modes.
- **Premium Design**: Glassmorphism and Dark Mode.

## Project Structure

- **backend/**: Java Spring Boot Application (API, File Processing, AI Orchestration)
- **frontend/**: React + Vite Application (UI, Visualization)

## Getting Started

### Prerequisites
- Java 17+
- Node.js 18+
- OpenAI API Key

### Running the App

1. **Backend**:
   ```bash
   cd backend
   .\mvnw.cmd spring-boot:run
   ```
   *Runs on port 8081*

2. **Frontend**:
   ```bash
   cd frontend
   npm install
   npm run dev
   ```
